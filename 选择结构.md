# C++支持最基本的三种程序运行结构：顺序结构，选择结构，循环结构
## 一、选择结构
### 1.单行 if 语句：if  (条件) {条件满足执行的语句}
<img width="1397" height="1280" alt="image" src="https://github.com/user-attachments/assets/800c8e55-a2ca-43ba-a7ed-33617e3128d8" />

>示例代码如下
```  
#include <iostream>
using namespace std;
int main()
{
    int a = 0;
    cout << "请输入一个分数：";
    cin >> a;
    if (a >= 90)
    {
        cout << "A" << endl;
    }
    return 0;
}
```
### 2.多行 if 语句：if (条件) {条件满足执行的语句} else {条件不满足执行的语句}
