# 一、什么叫常量
常量就是在程序运行过程中不可以改变的数值

# 二、常量表示

## 1.宏定义法：#define  常量名  常量值
（1）#define 与常量名中间不要加等号，会报错！    
（2）若使用宏定义法，则此代码写在文件的上方（讲白了就是写主函数前头，头文件底下）  

下面是一个运用宏定义的例子
```
#include <iostream>
using namespace std;

#define a 10
#define b 5

int main()
{
   int c;
   c = a * b;
   cout << c << endl;
   return 0;
}
```
>最后结果应为 50

## 2.const修饰法：const  数据类型 常量名 = 常量值  

注意：此法旨在在变量定义前加关键字const，把变量转化为常量

下面是一个运用const修饰的例子  
```
#include <iostream>
using namespace std;

int main()
{
   const int a = 15;
   const int b = 2;
   int c;
   c = a * b;
   cout << c << endl;
   return 0;
}
```
>最后结果应为 30
